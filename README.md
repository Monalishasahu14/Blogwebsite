# Python Blog Website

## Description
**Python Blog Website** is a modern, responsive web platform for sharing articles, tutorials, and personal stories. Built using **Python (Django/Flask), HTML, CSS, and JavaScript**, it focuses on clean design, smooth navigation, and user-friendly interface.  

Users can create, edit, and read blog posts. Admins can manage content efficiently. This project demonstrates full-stack development skills using Python for backend and React/HTML/CSS for frontend.

---

## Features
- User authentication (register, login, logout)
- Create, edit, and delete blog posts
- Comment system for posts
- Responsive design for mobile and desktop
- Admin dashboard for content management

---
